#### 1 分类

| 原生应用（native application，简称 native App）              | Web 应用（web application，简称 Web App）                    | 混合应用（hybrid application，简称 hybrid App）-h5           |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 原生 App 是专门为特定手机平台开发的应用程序 ，无法在其他平台运行。<br />iOS 的原生 App 用 Objective-C 或 Swift <br />安卓使用 Java 或 Kotlin | Web App 是使用网页做的应用程序，必须在浏览器中使用           | 原生 App 与 Web App 的结合。它的壳是原生 App，但是里面放的是网页 |
| （1）较好的性能和体验；<br />（2）可以使用系统的所有硬件和软件 API，比如 GPS、摄像头、麦克风等 | （1）不需要下载安装，打开浏览器就能使用，而且总是使用最新版本；<br />（2）对于开发者来说，Web App 写起来比较快，调试容易，不需要应用商店的批准就能发布。 | 跨平台、灵活性、开发方便                                     |
| （1）每个平台都要团队，成本高；<br />（2）开发、调试成本较高、周期长<br />（3）必须下载安装才能使用 | 1 不能从手机的首屏直接进入。<br />2 缺乏手机状态栏和锁屏时的通知推送能力。 <br />3不支持脱机访问（即断网也能使用）。 | 性能比较欠缺、可能比 Web App 都要慢一些                      |
|                                                              |                                                              |                                                              |
