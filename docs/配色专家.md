## 1 简介

## 2- 技术栈

项目基于 HTML5 + Scss + TypeScript + vue3



## 3- 使用 [vite搭建项目](https://v3.vuejs.org/guide/installation.html#vite)

### 3.1 开始

```js
$ npm init vite-app <awesome-color>
$ cd <awesome-color>
$ npm install
$ npm run dev
```

### 3.2 安装与配置TypeScript

安装：

```js
npm install -s typescript
```

配置：

1. `npx tsc --init` 创建`tsconfig.json`文件
2. 把根目录下的`main.js`文件改名为`main.ts`
3. 把根目录下的`index.html`中引入的`main.js`改名为`main.ts`
4. 同时把`.vue`文件里的`<script>`标签中加入`lang="ts"`
5. 在项目根目录创建`shim.d.ts`文件，同时在其中写入以下代码，用于配置ts支持识别`.vue`文件



### 3.3 安装与配置 router

安装：

```
npm install -s vue-router@next
```

配置：

1. 在`src`下建立`router`目录并在其中创建`index.ts`文件，并在其中写入：

   ```tsx
   import { RouteRecordRaw, createRouter, createWebHistory } from "vue-router";
   
   const routes: RouteRecordRaw[] = [
     {
       path: "/",
       name: "Home",
       component: import("./../views/index.vue"),
     },
   ];
   
   const router = createRouter({
     history: createWebHistory(process.env.BASE_URL),
     routes,
   });
   
   export default router;
   ```

2. 修改`main.ts`文件引入`vue-router`

   ```tsx
   import { createApp } from 'vue'
   import App from './App.vue'
   import './index.css'
   import router from "./router/index";
   
   createApp(App).mount('#app')
   ```
