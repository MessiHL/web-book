## 1 方法论

### 1.1 ROE（Return on Equity）: 净资产收益率

![](D:\doc\images\财经\EOR.png)



参考资料：[方法论三](https://mp.weixin.qq.com/s/L_h-p8wpzvEzorKgeXCu2g)

笔记：



投资一个行业，首先就要研究这个行业，在变化中寻找不变。

寻找要素：

1- 	影响行业变化的因素是什么？

2- 	这些因素导致行业发生了哪些变化？

3-	这些因素未来会不会变？能不能成为我们要找的变化中的不变？



如何寻找 影响行业变化的因素：

1 **通过上市公司披露的招股说明书和年报**

​	1- 先看100份年报，找感觉，了解行业

​	2- 尝试从业务去猜测公司财务情况

​	3- 重点关注年报的逻辑思路，不要只看结论



2 **查看上市公司的财务报表，找到异常波动的那年**

​	1- 尝试从财务报表去猜测公司的业务



**核心因素导致的 行业变化：**

1-市场规模、空间变化（如老人数量之于养老产业，牛奶之于猪肉）

2- 市场份额变化





**行业研究四部曲：**

1- 寻找影响行业变化的核心因素。阅读年报、研报、现场调研、专家访谈等，初始目标：先看100份年报

2- 确认自己的能力圈。确定核心因素变化时能否顶住

3- 研究这些核心因素对行业造成了哪些影响

4- 寻找可投资的。
