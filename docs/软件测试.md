#                                     软件测试

职位描述：

工作职责： 

1、根据需求文档，制定测试策略、编写测试计划；

 2、独立负责建立产品自动化测试框架、包括服务端APIWeb移动端的自动化测试工作，控制产品质量，提高团队测试效率； 

3、负责自动化测试脚本及辅助测试工具的开发，迭代优化测试框架； 

4、主导参与公司的大型项目的性能测试及调优工作，组织搭建公司级的自动化和性能测试平台. 



任职资格： 

1、精通软件测试理论和流程，掌握单元测试、集成测试、系统测试等各阶段测试所需要的原则、流程和方法； 

2、精通主流的自动化测试工具和测试框架，如selenium，Robot Rest-assured httpclient TestNG等，同时熟悉性能测试方法及常用工具，如Load Runnder、JMeter，Locust，具有丰富的实战经验； 

3、掌握Java，Python编程语言和具有相关编程经验; 熟悉Tomcat，mysql等主流中间件和数据库者优先; 

4、精通业界主流的CICD的工具 如Jenkins 熟悉Linux操作系统 精通Shell脚本的编写; 

5、具备云服务大数据系统算法测试经验者优先； 

6、较强的逻辑分析能力及解决问题能力；

 7、本科及以上学历，计算机及相关专业，7年以上测试工作经验。
